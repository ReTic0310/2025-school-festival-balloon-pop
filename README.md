# BALLOON SHOOTER 🎈🔫

USBカメラとハンドジェスチャーで遊ぶ、8ビットレトロスタイルのバルーンシューティングゲーム！

## 概要

MediaPipe Handsを使用したリアルタイムの銃ジェスチャー認識により、手で銃のポーズを作って風船を撃ち落とすゲームです。30秒間でできるだけ多くの風船を割って、ハイスコアを目指しましょう！

## 特徴

- 🎮 **ハンドジェスチャー認識**: 銃のポーズで照準を合わせて発射
- 🎯 **照準システム**: 銃を横に向けると照準が表示される
- 🔊 **8ビット風効果音**: 発砲音、風船破裂音、勝利/敗北音をリアルタイム再生
- 📷 **USBカメラ自動検出**: シリアル番号による自動デバイス認識（ポート変更に対応）
- 🎨 **8ビットレトロ風グラフィック**: 480x270の仮想解像度を1920x1080にスケール
- ⏱️ **タイムチケット報酬システム**: スコアに応じて報酬を獲得
- 📹 **リアルタイムカメラプレビュー**: 画面右上にカメラ映像を表示
- 🎈 **多様な風船タイプ**: 通常、ボーナス、ペナルティ、高速、ジグザグ、超レアの6種類

## システム要件

- Python 3.11 (MediaPipe互換性のため)
- USBカメラ (UVC対応推奨)
- Raspberry Pi または Linux システム
- ディスプレイ: 1920x1080 (フルスクリーン)

### 1. 依存ライブラリのインストール


## 使い方

### ゲームの起動

```bash
./main.py
```

または

```bash
./venv/bin/python3 main.py
```

### 操作方法

#### ゲーム開始前 (READYステート)
- **SPACE**: ゲーム開始

#### ゲーム中 (RUNステート)
- **銃ジェスチャー**:
  1. **照準モード**: 人差し指を伸ばして他の指を曲げ、手を横に向ける
     - 照準（黄色の円）が表示される
  2. **発射モード**: 照準を合わせたら、手を90度上に向ける
     - 照準位置で風船を撃ち抜く！
     - 🔊 発砲音が鳴る
     - 🎈 風船を割ると破裂音が鳴る
- **M**: マニュアル発射 (テスト用)
- **ESC / Q**: ゲーム終了

#### リザルト画面 (RESULTステート)
- **R**: リスタート
- **Q**: 終了

### 風船の種類

| 風船タイプ | 色 | スコア | 特徴 | 出現率 |
|----------|---|--------|------|--------|
| 通常 | カラフル | +10点 | 通常速度 | 35% |
| ボーナス | 金色 | +20点 | 少し大きい | 20% |
| ペナルティ | 紫色（ドクロマーク） | -30点 | 割ると減点！ | 20% |
| 高速 | カラフル（稲妻マーク） | +15点 | 1.8倍速 | 10% |
| ジグザグ | カラフル（波マーク） | +15点 | 左右に移動 | 10% |
| 超レア | マゼンタ（100マーク） | +100点 | 3倍速！ | 5% |

### タイムチケット報酬

ゲーム終了時、スコアに応じてタイムチケット（時間報酬）を獲得できます。

| スコア | 獲得時間 | 効果音 |
|--------|---------|--------|
| 350点以上 | +90秒（1分30秒） | 🎵 勝利音 |
| 325点 | +75秒（1分15秒） | 🎵 勝利音 |
| 275点 | +45秒 | 🎵 勝利音 |
| 225点 | +15秒 | 🎵 勝利音 |
| 200点 | 0秒（境界点） | 🎵 勝利音 |
| 200点未満 | マイナスペナルティ | 🎵 敗北音 |

### 効果音について

本ゲームでは8ビット風のレトロな効果音を採用しています：

- **🔫 発砲音 (shoot.wav)**: 銃を撃つたびに鳴る短いノイズバースト
- **🎈 破裂音 (pop.wav)**: 風船を割ったときに鳴る高音から低音へ下降する音
- **🎵 勝利音 (win.wav)**: ゲームクリア時の上昇アルペジオ（C-E-G-C）
- **💀 敗北音 (lose.wav)**: ゲームオーバー時の下降音階（G-F-D-C）

効果音は`generate_sounds.py`スクリプトで自動生成されます。音量は各効果音ごとに調整されており、ゲームプレイの邪魔にならないように設定されています。

## カメラ設定

初回起動時、USBカメラの情報が自動的に `camera_config.json` に記録されます。

```json
{
  "registered_cameras": [
    {
      "model": "C505 HD Webcam",
      "serial": "962EAD20",
      "preferred_resolution": [1280, 720],
      "preferred_fps": 30,
      "last_seen_device": "/dev/video0"
    }
  ]
}
```

次回以降は、カメラがどのポート（/dev/video0、/dev/video1など）に接続されていても、シリアル番号で自動的に認識されます。

## トラブルシューティング

### カメラが認識されない

```bash
# カメラデバイスの確認
v4l2-ctl --list-devices

# カメラの詳細情報
v4l2-ctl -d /dev/video0 --all
```

### Pygameの画面が表示されない

Raspberry Piの場合、X11またはWaylandディスプレイサーバーが実行されていることを確認してください。

```bash
echo $DISPLAY
# 出力例: :0 または :1
```

### MediaPipeのインポートエラー

Python 3.11を使用していることを確認してください：

```bash
./venv/bin/python3 --version
# Python 3.11.14 であることを確認
```

### 効果音が鳴らない

効果音ファイルが存在するか確認してください：

```bash
ls -l assets/sounds/
# shoot.wav, pop.wav, win.wav, lose.wav が存在することを確認
```

効果音ファイルが存在しない場合、生成スクリプトを実行してください：

```bash
./venv/bin/python3 generate_sounds.py
```

また、オーディオ出力デバイスが正しく設定されているか確認してください：

```bash
# オーディオデバイスの確認
aplay -l
```
